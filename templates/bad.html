Blend All Databases (BAD Tool)

Data blending is the process of combining data from multiple sources into a single, actionable dataset. With additional information from a secondary data source combined with data from the primary source, you see everything in one view. The process allows to gather insights from various sources and analyze them more deeply.

This tool also allowed to blend data from relational databases and non relational databases simultaneously.

Following Databases were Supported:-

  1. MongoDB
  2. MariaDB
  3. Postgres
  4. MySql
  5. Microsoft SQL Server
  6. Oracle Database
  7. SQLite
  8. CouchDB
